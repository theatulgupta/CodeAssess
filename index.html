1. Transpose of Matrix
33 Points
Problem Statement
Given a 2D matrix, return the transpose of the matrix. The transpose of a matrix is obtained by changing rows to columns and columns to rows.
Example
Input: matrix = [[1,2,3],[4,5,6],[7,8,9]]
Output: [[1,4,7],[2,5,8],[3,6,9]]
Explanation: Rows become columns and columns become rows.
Constraints
  • 1 ≤ matrix.length ≤ 1000
  • 1 ≤ matrix[i].length ≤ 1000
  • -109 ≤ matrix[i][j] ≤ 109
Sample Test Cases
Test 1: [[1,2,3],[4,5,6]] → [[1,4],[2,5],[3,6]]
Test 2: [[1,2],[3,4],[5,6]] → [[1,3,5],[2,4,6]]
Test 3: [[1]] → [[1]]
vector<vector<int>> transpose(vector<vector<int>>& matrix);
2. Merge Two Sorted Arrays
33 Points
Problem Statement
Given two sorted arrays nums1 and nums2, merge them into a single sorted array and return it. The merged array should be sorted in non-decreasing order.
Example
Input: nums1 = [1,2,3], nums2 = [2,5,6]
Output: [1,2,2,3,5,6]
Explanation: Merge both arrays maintaining sorted order.
Constraints
  • 0 ≤ nums1.length, nums2.length ≤ 200
  • 1 ≤ nums1.length + nums2.length ≤ 200
  • -109 ≤ nums1[i], nums2[j] ≤ 109
Sample Test Cases
Test 1: [1,2,3] + [2,5,6] → [1,2,2,3,5,6]
Test 2: [1] + [] → [1]
Test 3: [] + [1] → [1]
vector<int> mergeSortedArrays(vector<int>& nums1, vector<int>& nums2);
3. Intersection of Two Vectors
34 Points
Problem Statement
Given two integer vectors nums1 and nums2, return a vector of their intersection. Each element in the result must be unique and you may return the result in any order.
Example
Input: nums1 = [1,2,2,1], nums2 = [2,2]
Output: [2]
Explanation: Only 2 is common in both arrays.
Constraints
  • 1 ≤ nums1.length, nums2.length ≤ 1000
  • 0 ≤ nums1[i], nums2[i] ≤ 1000
  • Each element in result must be unique
Sample Test Cases
Test 1: [1,2,2,1] ∩ [2,2] → [2]
Test 2: [4,9,5] ∩ [9,4,9,8,4] → [4,9] or [9,4]
Test 3: [1,2,3] ∩ [4,5,6] → []
vector<int> intersection(vector<int>& nums1, vector<int>& nums2);
